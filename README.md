## kasync

### Inroduction to AsyncIO and Aiohttp

#### Features
* Ready to be hosted on Heroku
* GET and POST methods

#### Commands for Heroku Hosting
* $ heroku create
* $ git add .
* $ git commit -m "Asncio app"
* $ git push heroku master

#### Running app locally

* $python app.py

** Please note it's Python3.4.4  **
